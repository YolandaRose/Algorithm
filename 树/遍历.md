# 遍历

## 中序遍历

```java
public static List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<Integer>();
        inorder(root, res);
        return res;
    }
    public static void inorder(TreeNode root, List<Integer> res){
        if(root == null)return;
        inorder(root.left, res);
        res.add(root.val);
        inorder(root.right, res);
    }
```
